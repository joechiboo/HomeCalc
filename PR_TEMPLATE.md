# 🚀 HomeCalc MVP - 房貸試算器初版發布

## 📋 摘要

實作 HomeCalc 房貸試算器的 MVP 版本，包含完整的規格驅動開發（SDD）文件、核心計算功能、響應式 UI，以及 GitHub Pages 自動部署。

## ✨ 主要功能

### 1. SDD 文件建立
- ✅ **Constitution.md** - 開發憲章
  - 定義開發原則、技術棧、程式碼規範
  - 採用 Vue.js 3 + Vite + Chart.js 架構
- ✅ **Specification.md** - 產品規格
  - 完整的功能需求與使用者故事
  - 分階段實作計劃（MVP → 進階功能）
  - UI/UX 設計規範與測試案例
- ✅ **README.md** - 專案文檔
  - 快速開始指南與技術架構說明

### 2. 核心房貸試算功能
- ✅ **本息平均攤還計算**
  - 精確的還款期數計算（公式驗證）
  - 總還款金額與總利息計算
  - 每月本金/利息拆分明細
- ✅ **輸入參數**
  - 本金餘額（預設：16,102,679 元）
  - 年利率（預設：2.16%）
  - 每月還款金額（預設：64,432 元）
- ✅ **快速方案選擇**
  - 最低還款：64,432 元
  - 標準還款：70,000 元
  - 積極還款：80,000 元
- ✅ **方案比較功能**
  - 與標準方案（70,000/月，2.19%）自動對比
  - 顯示每月差額、期限差異、利息節省

### 3. UI/UX 設計
- ✅ **響應式設計** - 支援桌面、平板、手機
- ✅ **即時計算** - 輸入改變自動更新結果
- ✅ **漸層美化** - 現代化的視覺設計
- ✅ **卡片式佈局** - 清晰的資訊層級
- ✅ **數字格式化** - 千分位逗號顯示

### 4. GitHub Pages 部署
- ✅ **GitHub Actions** 自動部署 workflow
- ✅ **Vite 配置** - 正確的 base path
- ✅ **手動觸發** - 支援 workflow_dispatch
